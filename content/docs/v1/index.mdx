# fandom.js

fandom.js is a modern, typed JavaScript/TypeScript client for the Fandom (MediaWiki) APIs with ergonomic managers, robust rate limiting, and first-class event support. It is designed for reliability and clarity: predictable request flow, clear structures, and thorough type coverage.

## Highlights

- Strongly typed client and structures
- Composable managers for pages, revisions, users, search, categories, and meta
- Integrated request manager with retry and backoff
- Token-aware authentication flows
- Event system for reactive applications
- Built-in rate limiter for safe, concurrent workloads
