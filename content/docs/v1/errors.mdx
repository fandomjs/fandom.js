# Errors

All failures throw `ApiError` with:

- `code`: string | number
- `message`: string
- `status`: number
- `cause`: unknown | undefined

## Handling

```ts
try {
  const page = await client.pages.getById(123);
} catch (err) {
  if (err.name === 'ApiError') {
    // inspect err.status, err.code
  }
}
```
