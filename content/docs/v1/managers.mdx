# Managers Overview

Managers provide focused APIs over specific domains.

## List

- PageManager: CRUD and lookup of pages
- RevisionManager: Fetch and inspect revisions
- SearchManager: Site-wide querying
- UserManager: Lookup and metadata for users
- EventManager: Subscribe to events
- CategoryManager: Work with categories and hierarchies
- MetaManager: Site metadata and endpoints

Each manager exposes typed methods and returns typed structures. See individual pages for details.
